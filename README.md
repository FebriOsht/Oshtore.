## 🎨 Desain & Optimasi

```
Performance Highlights
──────────────────────────────────────────────
✓  Animasi blob → static (tidak ada infinite scale/opacity)
✓  Float animations pakai CSS will-change: transform
✓  AnimatedNumber → requestAnimationFrame (bukan setInterval)
✓  Semua data dipisah ke /data/*.ts untuk tree-shaking
✓  Texture lokal (/noise.svg, /cubes.png, /carbon-fibre.png)
✓  Watermark CSS via data:URI — tidak ada request external
✓  Navbar adaptive: collapse smart saat scroll
```

## 🔒 Proteksi Konten

Website ini dilengkapi sistem watermark berlapis:

```
Web Watermark
└── CSS background-image via data:URI SVG
    ├── Position: fixed (mengikuti scroll)
    ├── Mix-blend-mode: screen
    ├── Pointer-events: none
    └── Tidak bisa dihapus via DevTools inspect

PDF Watermark
└── ReportLab baked-in diagonal tiles
    ├── Teks "OSHTORE" + "oshtore.vercel.app"
    ├── Opacity rendah tapi terlihat di screenshot
    └── Tidak bisa dihapus via PDF editor biasa
```
